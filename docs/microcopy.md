# Microcopy

Vacíos:
- Clientes: “Aún no hay clientes — Crear cliente” (CTA primario)

Éxitos:
- Pago OK: “Pago recibido — Descargar CFDI o Volver a inicio”
- Ticket asignado: “Ticket asignado a [Técnico] — Ver ruta”

Errores:
- Checklist incompleto: banner bloqueante con “Ir al punto faltante”.
- Timeouts: “Tiempo de espera agotado. Reintentar.”

Toasts: 2.0s, aria-live="polite".
Modales: sólo confirmaciones destructivas; demás en sheets/sidepanels.
Validación inline: RFC, CP, tarjeta (PAN/exp/CSC) con ayudas.
Undo: SPEI (10 min), reasignación, borradores.
